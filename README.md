# BBC 데이터셋 샘플

<h2>1001개의 BBC 뉴스 기사로 구성된 샘플 データセット</h2>

![BBC dataset header](https://github.com/bright-kr/BBC-dataset-samples/blob/main/BBC-datasets.png)

1000개 이상의 레코드로 구성된 BBC データセット 샘플입니다. データセット는 <b>Bright Data API</b>를 사용하여 추출되었습니다.

<h2>BBC データセット에 포함된 일부 데이터 포인트:</h2>

* ```id```: 뉴스 기사의 고유 식별자
* ```url```: 기사가 게시된 웹 주소
* ```author```: 기사를 작성한 기자 또는 기고자의 이름
* ```headline```: 기사의 주요 제목
* ```topics```: 기사와 관련된 토픽 배열
* ```publication_date```: 기사가 게시된 날짜
* ```content```: 기사 전체 텍스트
* ```videos```: 기사와 관련된 임베드된 비디오
* ```images```: 기사에 포함된 이미지
* ```related_articles```: 토픽과 관련 있는 다른 기사로의 링크

그 외에도 더 많은 항목이 있습니다.

이는 "BBC news"
データセット에서 파생된 샘플 서브셋이며, 해당 データセット에는 <b>75K 레코드</b> 이상이 포함되어 있습니다.

사용 가능한 データセット 파일 형식: <b>JSON, NDJSON, JSON Lines, CSV, or Parquet. 선택적으로 파일을 .gz로 압축할 수 있습니다</b>.

データセット 전달 유형 옵션: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>.

업데이트 주기: <b>Once, Daily, Weekly, Monthly, Quarterly, or Custom basis</b>.

추출된 데이터 포인트에 추가로 제공 가능한 데이터 강화: <b>요청에 따라 제공됩니다.</b>

<b>[전체 BBC データセット 받기](https://brightdata.co.kr/products/datasets/bbc)</b>.

<h2>BBC データセット의 사용 사례는 무엇입니까?</h2>

<h3>1. Media Trend Analysis</h3>
BBC データセット를 사용하여 미디어 트렌드를 추적하고 시간 경과에 따른 뉴스 보도의 변화를 분석하며, 토픽 빈도와 프레이밍에 중점을 둡니다.

<h3>2.  Information Integrity</h3>
BBC データセット를 사용해 가짜 뉴스를 탐지하고 정보의 무결성을 평가하는 알고리즘을 개발하는 데 활용합니다.

<h3>3. Economic Forecasting</h3>
BBC データセット를 고도화된 알고리즘 트레이딩 모델 및 경제 예측 도구에 통합합니다. 실시간 뉴스 데이터를 트레이딩 알고리즘에 제공함으로써, 속보 이벤트, 경제 보고서 또는 정치적 전개로 인해 촉발되는 시장 변동에 이러한 시스템이 신속하고 효과적으로 대응할 수 있도록 하는 것이 목표입니다.
